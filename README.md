## これなに

スマートフォンのWeb版Mastodonを便利にする拡張機能です。  
タイムラインを見ながらトゥートできるように画面下にトゥート入力欄があります。  
不要なスペースをなくし、表示できるトゥート数を増やしています。  
Twitterクライアントの「SobaCha」のようにしたくて作成しました。
